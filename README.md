# 🎓 Student Details Management System

## 📌 Project Overview

The **Student Details Management System** is a Python-based application designed to manage student records efficiently. It provides a simple interface to store, view, search, update, and delete student information. This project demonstrates Python programming fundamentals, data handling, and CRUD (Create, Read, Update, Delete) operations.

---

## ✨ Features

* ➕ Add new student details
* 📋 View all student records
* 🔍 Search student by ID or Name
* ✏️ Update student information
* ❌ Delete student records
* 💾 Store and manage student data
* 🖥️ Simple and user-friendly console interface

## ▶️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-details-management-system.git
```

2. Navigate to the project folder:

```bash
cd student-details-management-system
```

3. Run the application:

```bash
python main.py
```

> If your system uses `python3`, run:

```bash
python3 main.py
```
